Privacy Policy — YouTube Fact Checker

Last updated: July 22, 2026

1. Data We Collect

The extension stores the following locally in your browser only:

DataPurpose
AI API KeyAuthenticate with OpenAI/Gemini/Claude
TranscriptsYouTube captions for analysis
Speaker TagsPer-video speaker assignments
Fact-Check ResultsAI verdicts and explanations
PreferencesProvider, model, theme settings

2. How We Store Data

All data lives in chrome.storage.local. No data hits our servers. No sync across devices.

3. Third-Party Services

Transcript text is sent to your chosen AI provider using your own API key. YouTube captions are fetched directly from googlevideo.com.

4. Data Retention

Uninstalling the extension deletes everything. You can also clear data via chrome://extensions without uninstalling.
